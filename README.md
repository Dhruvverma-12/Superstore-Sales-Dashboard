# Superstore Sales Analytics Dashboard | Power BI

## Project Overview

The **Superstore Sales Analytics Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to analyze retail sales performance across products, customers, categories, and states.

This dashboard transforms raw transactional data into meaningful insights, helping businesses monitor **sales trends, profitability, customer performance, and regional growth opportunities** through dynamic visualizations and KPI tracking.

The project demonstrates practical skills in:

- Data Cleaning
- Data Modeling
- DAX Calculations
- Dashboard Design
- Business Intelligence
- Data Visualization
- Data Storytelling

## Business Objectives

This dashboard helps answer important business questions:

✔️ Which states generate the highest sales revenue?

✔️ Which products contribute the most sales?

✔️ Which sub-categories generate maximum profit?

✔️ How do monthly sales and profits vary?

✔️ Which customers contribute the highest profit?

✔️ What is the category-wise quantity distribution?

✔️ Which quarter performs best?

# Key Insights

### Revenue Insights

- California generates the highest sales.
- A small group of products drives a significant portion of revenue.

### Profitability Insights

- Copiers contribute the highest profit.
- High sales do not always result in high profits.

### Geographic Insights

- Revenue is concentrated in a few states.
- Several regions offer growth opportunities.

### Customer Insights

- Top customers contribute disproportionately to profits.
- Customer-focused strategies can improve retention and revenue.
